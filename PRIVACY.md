# Privacy Policy — inkling

Last updated: 2026-04-19

## Summary

inkling does not collect, store, or transmit any personal data. All user preferences are stored locally on your device.

## Data collection

inkling does not collect any data. Specifically:

- No personal information is collected
- No browsing history is tracked
- No page content is transmitted to any server
- No analytics or telemetry is used
- No cookies are set

## Local storage

inkling stores the following preferences locally on your device using Chrome's built-in storage API (`chrome.storage.local`):

- Display mode selection (e.g., background, underline)
- Global enabled/disabled state
- List of domains where the extension is disabled

This data never leaves your device.

## Permissions

- **storage**: Save user preferences locally
- **activeTab**: Read the current tab's hostname when the popup is opened, for per-domain settings
- **Host permissions (all URLs)**: Run the content script that detects and highlights color codes in page text

## Third-party services

inkling does not communicate with any external servers or third-party services.

If this policy changes, the updated version will be posted at this URL.
